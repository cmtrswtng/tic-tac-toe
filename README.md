## Установка и запуск

### Требования
- Node.js
- npm или yarn

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```
Приложение будет доступно по адресу http://localhost:5173

### Сборка для продакшена
```bash
npm run build
```

### Предварительный просмотр сборки
```bash
npm run preview
```

### Проверка кода линтером
```bash
npm run lint
```
